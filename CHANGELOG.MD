# v1.0.0

## Added

- Integração completa entre Frontend e Backend.
- Consumo da API REST através do endpoint GET /dashboard.
- Dashboard alimentado por dados reais do MySQL.
- Remoção da dependência do Mock Data.
- Comunicação entre Frontend (JavaScript) e Backend (Node.js + Express).
- Renderização dinâmica da fila de produção.
- Atualização automática dos cards com dados da API.

## Improved

- Arquitetura do Frontend reorganizada.
- Separação entre API, Dashboard, Render e Inicialização.
- Código preparado para futuras integrações com Login e Administração.

## Status

MVP integrado e funcional.
